# Laswell Robotics

A React + Vite + TypeScript app for showcasing Laswell Robotics services and products.

## Getting Started

```sh
npm install
npm run dev
```

## Tech Stack
- Vite
- React (TypeScript)
- Tailwind CSS + shadcn/ui

## Notes
- Place your logo at `public/logo.png` to be used across the app (Open Graph + header).
- Update metadata in `index.html` as needed.
